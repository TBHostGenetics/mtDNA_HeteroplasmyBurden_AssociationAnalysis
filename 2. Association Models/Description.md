**These scripts do the following steps:**

_1. AncestryRegression.R_
Uses negative binomial regression models to determine if sex, age, mean read depth or broad ancestry categories influences heteroplasmic and/or homoplasmic variant burden. 
This script also generates forest plots from results. 

_2. HaplogroupRegression.R_
Uses negative binomial regression models to determine if sex, age, mean read depth or mitochondrial haplogroups influences heteroplasmic and/or homoplasmic variant burden. 
This script also generates forest plots from results.
